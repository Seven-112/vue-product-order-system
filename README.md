# Vue Product Order System

## Project setup

```
git clone https://github.com/RakibDevs/vue-product-order-system.git
cd vue-product-order-system
cp .env.example .env
npm install
```
### API Endpoint
Set API endpoint in .env
```
VUE_APP_API_ENDPOINT=http://127.0.0.1:8000/api/v1/
```
